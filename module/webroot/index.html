导入节点

兼容分享链接、base64 订阅正文、sing-box JSON/outbounds 导出,支持 UTF-8 中文/emoji 节点名,自动转换成 sing-box 出站并写入下方 outbounds.json。支持 vmess / vless / trojan / ss / hysteria2 / tuic / anytls,以及 sing-box-extended 的 XHTTP。

替换:用导入结果整体替换节点列表;追加:保留已有节点,再并入新节点(同名自动改名)。

outbounds.json

出站节点配置(协议、传输层、TLS、多节点选优等)。保存前会先做本地 JSON 格式校验。

per-app 名单

决定哪些应用的流量进入 sing-box;未列入代理范围的应用完全不经过 tun,数据和 DNS 都走系统默认网络。

白名单只让点选应用进入 TUN,性能/省电最好;黑名单适合临时全局接管。

应用选择器

名单决定哪些应用进入 TUN;自动/代理/免流决定进入 TUN 后怎么出站。
